Application mobile « AI »

Ce projet personnel permet à des utilisateurs iPhone ou Android de prendre une photo et d'afficher «l'âge estimé» grâce à un algorithme de Machine Learning.

L'application se connecte à un API REST : L'API de Microsoft Azure Cognitive services.

Technologies utilisées: Xamarin.forms, C#, .NET, Azure Cognitive services, HTTP, JSON
